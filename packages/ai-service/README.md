# AI Service

Este serviço é o **AI Service**, responsável por toda a lógica de inteligência artificial do ZapFlow, usando o framework Genkit do Google.

## Responsabilidades

-   **RAG (Retrieval-Augmented Generation):** Indexar documentos e buscar informações relevantes para responder às perguntas dos clientes.
-   **Geração de Linguagem Natural:** Utilizar modelos de linguagem (como o Gemini) para gerar respostas inteligentes e personalizadas.

Para mais detalhes sobre a arquitetura e como este serviço se integra com os outros, consulte o [documento de arquitetura principal](../../ARCHITECTURE.md).
