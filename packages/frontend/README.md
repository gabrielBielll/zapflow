# Frontend

Este serviço é o **Painel de Controle (Frontend)** do ZapFlow, construído com Next.js e TypeScript.

## Responsabilidades

-   Fornecer a interface de gerenciamento para o dono do negócio.
-   Exibir o QR Code para conexão com o WhatsApp.
-   Permitir o upload de documentos para treinar a IA.
-   Exibir o histórico de conversas e as métricas de desempenho.

Para mais detalhes sobre a arquitetura e como este serviço se integra com os outros, consulte o [documento de arquitetura principal](../../ARCHITECTURE.md).
