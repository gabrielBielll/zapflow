# ZapFlow

ZapFlow é uma plataforma para criar assistentes virtuais inteligentes para WhatsApp, com foco em automação de vendas e atendimento ao cliente.

## Documentação

Para entender o projeto, comece com os seguintes documentos:

-   **[CHECKPOINT.md](./docs/CHECKPOINT.md):** Um resumo do estado atual de desenvolvimento, o que está pronto e quais são os próximos passos.
-   **[Guia de Usuário (GUIDE.md)](./GUIDE.md):** Instruções sobre como configurar e executar o projeto.
-   **[Arquitetura do Projeto (ARCHITECTURE.md)](./ARCHITECTURE.md):** Uma visão geral da arquitetura de microsserviços do ZapFlow.
-   **[Guia de Implantação (DEPLOYMENT.md)](./DEPLOYMENT.md):** Instruções sobre como fazer o deploy da aplicação.
