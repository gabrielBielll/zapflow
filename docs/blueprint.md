# **App Name**: ZapFlow Frontend

## Core Features:

- Chatbot Type Selection: Initial screen to select the chatbot type (AI Generative, Flow-based, Automated Triage).
- Chatbot Creation Form: Form for initial chatbot configuration (name, purpose) for the AI Generative chatbot.
- Main Configuration Dashboard: Main dashboard with navigation for Personality, Knowledge Base, Skills, and Channels.
- Personality Customization: Allows the LLM tool to define personality traits of the chatbot including tone and style based on a form with range sliders and checkboxes.
- Knowledge Base Management: Displays and manages the chatbot's knowledge base (PDFs, DOCXs) with a grid view and upload functionality.
- Skill Management: Enables the management of chatbot skills (FAQs, Transfer to Human) with toggle switches to activate/deactivate.
- Channel Management: Manages communication channels for the chatbot. Includes an empty state and a link to connect new channels.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a clean and modern aesthetic.
- Background color: Light gray (#F9FAFB) to provide a subtle backdrop for content.
- Accent color: Teal (#329F91) to add visual interest and highlight interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, readable feel.
- Use minimalist, line-based icons to complement the modern design.
- Use white cards with subtle shadows for panels to create a sense of depth and hierarchy.
